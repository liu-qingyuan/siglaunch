# Triage 标签

| mattpocock/skills role | GitHub 标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完全说明，AFK agent 可以接手 |
| `ready-for-human` | `ready-for-human` | 需要人工实施 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个 triage role 时，使用表中对应的 GitHub 标签。
